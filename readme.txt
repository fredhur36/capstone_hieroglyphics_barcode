이 프로그램을 작동시키기 위해서는 컴퓨터에 python3 와 opencv-python이 설치되어 있어야 합니다. 
만약 설치되어있지 않다면, python3를 설치한후
windows: 커맨드 입력창에 pip3 install opencv-python 을 입력하세요
macOS  : 터미널 창을 열고, pip3 install opencv-python 을 입력하세요

프로그램 실행방법은 우선 macOS 에서만 설명하겠습니다.

1. compareImages.py 스크립트와 비교할 두 이미지를 같은 폴더에 넣습니다.
2. 터미널창을 열고, 위의 폴더로 이동합니다.
3. 만약 두 이미지의 이름이 test1.jpg test2.jpg 라면 
   터미널에다가 python3 compareImags.py test1.jpg test2.jpg 를 입력합니다.
4. 종료하려면 키보드 q를 입력하고 엔터를 칩니다.
